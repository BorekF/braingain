# BrainGain

An AI-assisted learning platform: admins add YouTube or PDF materials; students take AI-generated quizzes and earn reward minutes after passing.

**Stack:** Next.js (App Router), React 19, Supabase, OpenAI (GPT-4o-mini), Tailwind CSS. Server Actions handle quiz generation and materials; optional Groq for audio transcription.

## Live demo

**[View demo →](https://noncircuited-unabatingly-sommer.ngrok-free.dev/)**

The student dashboard is public. Quizzes are generated by GPT-4o-mini from the material transcript or PDF text. Questions are cached after first generation; a per-IP rate limit applies in demo mode.

## Getting started

```bash
npm install
```

Copy `.env.example` to `.env.local` and set the required variables (Supabase, OpenAI, admin secret). See `.env.example` for full list and comments.

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Root redirects to `/student`.

## Deployment

Deploy as a standard Next.js app (Vercel, Railway, etc.). Set all production environment variables; ensure `NODE_ENV=production` and `ADMIN_SECRET` is strong.
